About The Project (Flashcard)
This project is created mainly with React.js. In this project, we have three pages:
1) Create a Flashcard page  
2) My Flashcard page  
3) Flashcard Details page

• Create Flashcard Page: This page uses Formik for input boxes, Yup for validation, and a switch to toggle between light and dark modes.
• My Flashcard Page: Here, you can see all your flashcards.
• Flashcard Details Page: This page displays the details of your flashcards and allows you to share or print them.
How To Use
1. Open the flashcard generator Click Here.
2. Fill in all input boxes without any errors and click the “Create” button.
3. To view recently created flashcards, click on the “My Flashcard” link, which shows all flashcards you created recently.
4. To view the details of a flashcard, click the “View Card” button to be redirected to the flashcard details page.
